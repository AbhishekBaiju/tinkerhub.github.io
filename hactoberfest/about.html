How does the Mentorship Program Work?

You can start with filling this form. Upon submission we will send you a link to our discord channel where you can communicate with mentors and other participants about all things open source and hacktoberfest !

Our mentors are open source enthusiasts who hacked hacktoberfest in previous years. They are on a mission to help you to do the four pull requests and bring the open source community a new contributor by doing so. They will help you, to find interesting projects you can contribute, to learn from the road blocks you face in the journey and fix it and moreover to create an awesome hacktoberfest experience together.

Let the hacking begin !

What is actually HacktoberFest?

Hacktoberfest is a month-long celebration of open source software run by DigitalOcean in partnership with GitHub and Twilio. Hacktoberfest is open to everyone in our global community! Five quality pull requests must be submitted to public GitHub repositories.

To earn your Hacktoberfest tee or tree reward, you must register and make four valid pull requests (PRs) between October 1-31 (in any time zone).
